La nostra història

Nascut a l'Institut Tecnològic de Barcelona, Espurna és un projecte acadèmic que aplica tecnologia IoT real per resoldre un problema crític: la detecció ultra-precoç d'incendis forestals al Parc de Collserola.

La nostra missió

Protegir el pulmó verd de Barcelona detectant qualsevol risc d'incendi hores abans que es propagui. Els incendis forestals destrueixen milions d'hectàrees cada any — Espurna vol canviar això.

La nostra tecnologia

Sensors RAK4631 LoRaWAN amb cobertura de 15km transmeten dades de temperatura i humitat del sòl cada hora al dashboard central, amb alertes en temps real i visualització NASA FIRMS.

// Tecnologies

Tecnologies

Nodes RAK4631

Nordic nRF52840 + SX1262 LoRa. Sensors de temperatura, humitat i bateria. Activació OTAA amb AES-128.

Gateway RAK7289V2

8 canals EU868. Backhaul 4G/Ethernet. Desplegat a la coberta de l'ITB amb visibilitat al Collserola.

ChirpStack v4

Network Server LoRaWAN open-source. Gestió OTAA, decodificació de payloads i integració HTTP → Node-RED.

Node-RED + InfluxDB v2

Pipelines visuals de dades. Transformació, filtrat i emmagatzematge de sèries temporals amb Flux queries.

Dashboard Leaflet/Cesium

Mapa 2D/3D. NASA FIRMS, NDVI, EONET. Vent en viu, LOS, cobertura antena i gràfics Chart.js.

MeshCore

Xarxa mesh complementària amb 2 gateways, 2 repetidors i 2 sensors per zones sense cobertura directa.
