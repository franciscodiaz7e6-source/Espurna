## HARDWARE 
# ESP32 HELTEC LORA V4 (NODO MESHTASTIC)

# ESP32 GENERIC (NODO PROVA SENSORES)

# SENSORS BOSCH BME690 
Este es un sensor avanzado de medición de temperatura, humedad, calidad de aire, presión y gases 
Usa el protocolo I2C, SPI para comunicarse

Librerias Arduino:
1. 7Semi (https://github.com/7semi-solutions/7Semi-BME690-Arduino-Library/tree/main)
2. Bosch (https://github.com/boschsensortec/BME690_SensorAPI/tree/master)

Per ara farem servir la llibreria 7Semi, si en el futur veiem que es necesari el ús de la llibreria de Bosch, la considerarem.
